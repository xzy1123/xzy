## 新产品 / 新工具

### Jupitrr AI：全链路视频工作台 VideoOS
[Product Hunt](https://www.producthunt.com/products/jupitrr)  
→ 为什么值得关注：一站式视频制作，适合做营销、培训、教程的团队。**痛点：不用再在多个工具间来回切换，AI 自动化脚本→字幕→成片。**

### MailToDock：用 AI 把 Gmail 变成 Google Tasks
[Product Hunt](https://www.producthunt.com/products/mailtodock)  
→ 为什么值得关注：自动提取邮件中的待办事项并创建任务。**效率神器：每天处理上百封邮件的打工人强烈推荐。**

### AssemblyAI Voice Agent API：一句话构建生产级语音 Agent
[Product Hunt](https://www.producthunt.com/products/assemblyai)  
→ 为什么值得关注：一条 API 支持语音对话 Agent，含语音识别、合成、意图理解。**落地最快：呼叫中心、语音助手、客服机器人可以 2 周内上线。**

### Tinfoil：完全私密的 AI 聊天与 API
[Product Hunt](https://www.producthunt.com/products/tinfoil-private-ai-chat)  
→ 为什么值得关注：主打隐私，对话和 API 请求不会离开本地或加密传输。**适用场景：金融、医疗、法律等对数据安全要求极高的行业。**

### Rova AI：面向 Web & 移动应用的自主测试 Agent
[Product Hunt](https://www.producthunt.com/products/rova-ai)  
→ 为什么值得关注：像真实用户一样自主探索应用并发现 bug。**QA 团队福音：减少手工测试用例维护，让 AI 替你跑回归。**

### Hera Launch：用 AI 生成工作室级产品发布视频
[Product Hunt](https://www.producthunt.com/products/hera-6)  
→ 为什么值得关注：输入素材，AI 自动剪辑、配音、特效。**创业团队省预算，无需专业视频团队也能做出像样的宣发视频。**

## 开源项目 / 工程落地

### everything-claude-code：Agent 性能优化系统

→ 为什么值得关注：日增几千星，号称 Claude Code 的"技能、直觉、记忆、安全"全面增强。**适合：希望让 AI Agent 更稳定、更聪明的开发者，相当于给 Agent 装了一个自动驾驶系统。**

### microsoft/ai-agents-for-beginners：12 节课入门 AI Agent
[GitHub 6万星](https://github.com/microsoft/ai-agents-for-beginners)  
→ 为什么值得关注：微软官方出品，从零到一教你构建 Agent。**适合：想系统学习 Agent 开发的团队或个人，完全免费且附带实践项目。**

### Cherry Studio：300+ 助手 + 自主 Agent 的 AI 工作台
[GitHub 4.5万星](https://github.com/CherryHQ/cherry-studio)  
→ 为什么值得关注：统一接入主流大模型，内置自主 Agent 和数百个预设助手。**亮点：支持本地部署，适合企业搭建内部 AI 工作台。**

### MindsDB：AI Data Vault，让 Agent 安全查询任何数据源
[GitHub 3.9万星](https://github.com/mindsdb/mindsdb)  
→ 为什么值得关注：把数据库、API、文件系统整合成一个查询引擎，Agent 通过它安全读数据。**落地场景：数据中台 + AI Agent = 实时问答分析平台。**

### SGLang：高性能大模型 / 多模态模型服务框架
[GitHub 2.7万星](https://github.com/sgl-project/sglang)  
→ 为什么值得关注：专为推理优化，支持多模态（文本+图像+音频）。**工程价值：比标准 vLLM 在特定场景下吞吐更高，适合高并发 API。**

### Mastra：TypeScript 框架构建 AI 应用和 Agent
[GitHub 2.3万星](https://github.com/mastra-ai/mastra)  
→ 为什么值得关注：Gatsby 团队出品，面向现代 TypeScript 栈。**适合：前端/全栈开发者快速构建 AI 应用，无需切换语言。**
