## 🔧 开源项目 / 工程落地

**Milvus：高性能云原生向量数据库**  
[github.com/milvus-io/milvus](https://github.com/milvus-io/milvus)  
★ 44045 stars  
→ 为什么关注：向量数据库是知识库和 RAG 的核心，Milvus 是生产验证过的方案。最新版本性能又有提升，适合做大规模搜索增强。

**AionUi：免费开源 24/7 协作者**  
[github.com/iOfficeAI/AionUi](https://github.com/iOfficeAI/AionUi)  
★ 22896 stars  
→ 为什么关注：一个本地运行的开源“同工作”应用，同时支持 Gemini、Claude、Codex、Qwen 等多个 agent 接入。团队协作和混搭 agent 的好工具。

**openfang：开源 Agent 操作系统**  
[github.com/RightNow-AI/openfang](https://github.com/RightNow-AI/openfang)  
★ 17069 stars  
→ 为什么关注：目标是成为 agent 的底层操作系统，帮你管理 agent 生命周期、通信和资源。星星增长很快，说明大家都在找 agent 编排方案。

**ppt-master：AI 生成原生 PPTX**  
[github.com/hugohe3/ppt-master](https://github.com/hugohe3/ppt-master)  
★ 9381 stars  
→ 为什么关注：不是生成图片式的 PPT，而是输出原生 PowerPoint 形状，可以直接编辑。无需设计技能，文档一键变幻灯片。

**kreuzberg：多语言文档智能框架**  
[github.com/kreuzberg-dev/kreuzberg](https://github.com/kreuzberg-dev/kreuzberg)  
★ 8135 stars  
→ 为什么关注：Rust 内核，能从 PDF、Office 文档等提取文本、元数据、图片和结构化信息。处理文档类应用的基础设施。

**open-design：本地优先的 Claude Design 替代品**  
[github.com/nexu-io/open-design](https://github.com/nexu-io/open-design)  
★ 3184 stars  
→ 为什么关注：19 种技能、71 个品牌级设计系统、沙盒预览，全部本地运行。不想用 Anthropic 云服务的设计团队可以自建。
